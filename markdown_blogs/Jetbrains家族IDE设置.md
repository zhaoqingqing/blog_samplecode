### 单行字符最大宽度

Editor - Code Style - C#- Wrapping and Braces ,修改 Hard wrap 

Editor - Code Style - Python - Wrapping and Braces ,修改 Hard wrap at

### 统一快捷键

设置Pycharm的快捷键和Rider/Resharper一致，减少学习成本。

目前这两款IDE大多数的快捷键是统一的，比如Alt+F7 查找引用
